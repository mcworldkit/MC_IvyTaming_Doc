Pxr Standard Environment Map Light object node

Pxr Standard Environment Map Light object for RenderMan RIS.

A light object used for RenderMan’s RIS rendering. It represents an Pxr Standard Environment Map Light and wraps a corresponding RIS light shader as dive-in target.
